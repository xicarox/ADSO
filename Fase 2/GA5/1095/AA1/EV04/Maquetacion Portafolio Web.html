Proyectos

  • Scraping web

    El web scraping es una técnica de extracción de datos que se utiliza para obtener información de páginas web de forma automatizada. Esto implica escribir un programa informático que recopile la información deseada de un sitio web y la almacene en un formato estructurado y legible para su posterior procesamiento o análisis..

  • Automatizacion con ETL

    ETL (Extract, Transform, Load) es un proceso utilizado en el mundo de la tecnología de la información para la integración y gestión de datos. Se trata de un proceso que se utiliza para la extracción, transformación y carga de datos de diferentes fuentes a un destino final, como un data warehouse, una base de datos o una aplicación de análisis de datos.

  • Analisis de Datos con Power Bi

    Power BI es una plataforma de análisis de datos empresariales desarrollada por Microsoft. Permite a los usuarios conectarse a una amplia variedad de fuentes de datos, incluyendo bases de datos, archivos planos, servicios en la nube y aplicaciones de terceros, y crear informes interactivos, paneles de control y visualizaciones de datos personalizadas.
